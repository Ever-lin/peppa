本集语法讲解
📌语法一：If 零条件句
句型结构
If + 主语 + 动词原形，主语 + must / can + 动词原形
用法
表示"只要做了前面的事，就一定/必须发生后面的事"，常用于规则和惯例。
If you jump in muddy puddles, you must wear your boots.
如果跳泥坑 → 必须穿靴子（规定，无讨论余地）
举一反三
If it rains, we cannot play outside. 如果下雨，就不能出去玩。
If you are tired, you must rest. 如果累了，就必须休息。
📌语法二：现在完成时 have/has + 过去分词
句型结构
主语 + have / has + 过去分词
用法
过去发生的动作对现在有影响。第三人称单数用 has，其余用 have。
It's stopped raining. 雨停了 → 现在可以出门
Peppa has found a little puddle. 找到了 → 就在那里
举一反三
I have finished my homework. 我做完作业了。
She has eaten lunch. 她吃过午饭了。
📌语法三：现在完成进行时 have been + doing
句型结构
主语 + have / has been + 动词 -ing
用法
强调从过去某时开始、持续到现在的动作，结果往往可见。
You've been jumping in muddy puddles!
一直在跳 → 满身是泥（结果显而易见）
与完成时的区别
I have read the book. 读完了（强调完成）
I have been reading the book. 一直在读（强调持续）
📌语法四：情态动词 must / cannot
句型结构
must + 动词原形 → 必须（义务/规定）
cannot + 动词原形 → 不能（禁止/不允许）
小提示
情态动词后永远接动词原形，不加 to，不加 -s。
✅ must wear   ✅ cannot play
❌ must to wear   ❌ cannot plays
本集固定句型
💬6个超实用日常句型
句型① 打招呼 · 自我介绍
I'm ___. This is my ___, ___.
I'm Peppa Pig. This is my little brother, George. → 我是佩奇，这是我弟弟乔治。
I'm Tom. This is my best friend, Lucy. → 我是汤姆，这是我好朋友露西。
句型② 请求许可 · Can we…?
Can we go out / come in / have ___?
Can we go out to play? → 我们可以出去玩吗？
Can we have some ice cream? → 我们可以吃冰淇淋吗？
Can we come in? → 我们可以进来吗？
句型③ 发出邀请 · Let's…!
Let's find / go / try / play ___!
Let's find some more puddles! → 我们去找更多泥坑吧！
Let's clean up quickly! → 我们快点收拾吧！
Let's go to the park! → 我们去公园吧！
句型④ 猜测提问 · Guess what…!
Guess what I / we did / found / made!
Guess what we've been doing! → 猜猜我们刚才在做什么！
Guess what I found! → 猜猜我发现了什么！
Guess what I made for dinner! → 猜猜我做了什么晚餐！
句型⑤ 轻描淡写 · It's only…
It's only mud / water / a joke / a scratch.
Oh well, it's only mud. → 哦好吧，不过是泥巴嘛。
Don't cry, it's only a scratch. → 别哭，不过一点擦伤。
It's only a joke! → 不过是个玩笑而已！
句型⑥ 表示惊讶 · Goodness me!
Goodness me! / Oh my! / Dear me!
Goodness me! → 天哪！（英式，语气温和）
Oh my goodness! → 我的天哪！（程度更强）
💡 比 "Oh my God" 更礼貌，更英式日常。
